# Manual Ground Truth (OCR Evaluation)

This folder contains a manually curated index of document-level characteristics used to support OCR accuracy evaluation.

## Contents

- `jfk_files_index_manual.xlsx` – Spreadsheet with manually recorded metadata per document (page count, document type, OCR difficulty, etc.)

- `pdf files/` – 31 sample PDFs used for OCR evaluation
- `Fwd_ JFK documents/` – Manually transcribed text files for those documents
- `Fwd_ JFK documents.zip` – Zipped archive of the transcriptions
